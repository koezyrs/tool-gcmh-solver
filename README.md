# Tool Giải GCMH Gunny

Tool hỗ trợ giải Minesweeper cho GCMH Gunny.

## Triển khai trên Vercel

Dự án này là một web tĩnh (HTML/CSS/JS) và hoàn toàn tương thích với Vercel.

### Cách deploy:
1. Push code lên GitHub/GitLab/Bitbucket.
2. Vào [Vercel](https://vercel.com), chọn "New Project".
3. Import repository này.
4. Nhấn **Deploy**. Vercel sẽ tự động nhận diện và build.

### Cấu trúc
- `index.html`: File chính.
- `style.css`: Style (Gunny Zing theme).
- `script.js`: Logic giải thuật.
- `vercel.json`: Cấu hình Vercel.
